# Appium 自动化项目文档

本目录用于保存 Appium 自动化脚本生成项目的规划文档和阶段文档。

这些文档作为项目事实来源。后续如果方案发生变化，先修改对应文档；实现时以最新文档内容为准。

## 文档结构

- `plan.md`：整体规划，包括目标、范围、架构、阶段、风险和待决事项。
- `phase_01_poc/plan.md`：第一阶段 PoC 的实施细节。
- `phase_02_framework/plan.md`：第二阶段框架化提取计划。
- `phase_03_batch_generation/plan.md`：第三阶段批量生成计划。
